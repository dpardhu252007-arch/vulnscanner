# vulnscanner
Python-based Vulnerability Scanner for basic security assessments. Features include open port scanning, service detection, banner grabbing, outdated software identification, automated vulnerability reporting, and an interactive terminal interface. Built using Python, socket programming, and vulnerability assessment concepts.

#Features
Open Port Scanning
Service Detection and Banner Grabbing
Basic Vulnerability Identification
Automated Report Generation
User-Friendly Terminal Interface
Animated Startup Banner

#Technologies Used
Python 3
Socket Programming
Network Security Concepts
Vulnerability Assessment Fundamentals
